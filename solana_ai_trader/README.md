# solana_ai_trader

Projeto iniciado: espaço para um trader baseado em Solana com componentes de AI.

Estrutura sugerida:

- `bot.py` — entrypoint do bot
- `agent.py` — lógica do agente/AI
- `trader.py` — integração com a exchange / carteira
- `watcher.py` — observadores de mercado
- `utils.py` — utilitários e helpers

Como começar (exemplo):

```bash
cd solana_ai_trader
python -m venv .venv
source .venv/bin/activate
# adicionar dependências ao requirements.txt
pip install -r requirements.txt
```
